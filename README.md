## Salord-Real-Estate
## Overview
This is a modern, fully responsive landing page built using **HTML5, CSS3, and Bootstrap 5**. The project is designed with a mobile-first approach, ensuring a seamless user experience across all devices, including desktops, tablets, and smartphones.Please note that this website is a clone built entirely for practice.
- ## Features
  - Fully Responsive Design: Optimized for all screen sizes using Bootstrap's fluid grid system.
  - Modern UI/UX: Clean, pixel-perfect layout with intuitive navigation.
  - Interactive Elements: Includes dynamic components such as carousels, accordions, and responsive navigation bars.
  - Cross-Browser Compatibility: Works smoothly across Chrome, Firefox, Safari, and Edge.
